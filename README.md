# EcomXcape 🚀

EcomXcape is a modern, high-performance web application designed to help businesses scale through eCommerce automation. Built with React and Vite, it features a sleek, premium dark-themed UI that focuses on conversions, trust-building, and dynamic content presentation.

## 🌟 Key Features

- **Premium Design System**: Dark charcoal themes (`#1e1e1e`) accented by deep forest green (`#1a4d2e`) for a professional, trustworthy aesthetic.
- **Glassmorphism UI**: Modern frosted glass effects and subtle layering for 3D depth.
- **Interactive Components**: Smooth micro-animations, hover effects, and visually engaging trust metrics.
- **Responsive Layout**: Fully optimized for both desktop and mobile viewing.
- **High Performance**: Built on top of Vite for blazing fast Hot Module Replacement (HMR) and optimized production builds.

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Styling**: Modern Vanilla CSS with CSS Modules / Inline Styles

## 🚀 Getting Started

Follow these steps to set up the project locally.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Zu1qrnain/EcomXcape.git
   cd EcomXcape
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## 📦 Building for Production

To create an optimized production build:
```bash
npm run build
```
This will generate a `dist` folder containing the minified and optimized assets ready for deployment. To preview the production build locally:
```bash
npm run preview
```
